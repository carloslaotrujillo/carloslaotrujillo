Instructions

The url https://rma.swissgear.com/api/interview-endpoint was created for an interview test.

Below are query params for the route:

  • page - optional - integer starts at one - default: 1
  • sortBy - optional - name, price only valid options - default: "name"
  • direction - optional - asc, desc only valid options - default: "asc"
  • q - optional - string query for product name or sku
Test Description:

Create a html/css/js demo that displays products from our feed. The goal is to create a page that displays products that mostly matches styling of our existing category pages (example: https://www.swissgear.com/backpacks).

Details:

  • You don't need the SwissGear site header and footer, just the body part of the page design.
  • You don't need to include the Filters or Sort By dropdowns. However creating your own search input and price sorting would award bonus points.
  • Pagination is encouraged but not required if implemented we suggest using an infinite scroll implementation.
  • Color swatches and review stars not required.
